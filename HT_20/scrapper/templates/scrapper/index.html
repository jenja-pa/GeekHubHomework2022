Опис проекта

Домашнє завдання №19 - Django застосунок, скрепер та показ зібраної інформації

  1. Сторінка ("Додати продукти"), на якій Розміщено textfield, в який можна ввести будь-яку кількість ID продуктів.
  2. При відправленні форми, запускається скрейпер, що збирає всі продукти із визначеними ID із сайту Розетки.
  3. Процес скрейпінга відбувається "на бекграунді" - основний потік виконання не блокується:
    • В фоновий потік передається розібраний список отриманий із textfield, без застосування передічі через модель.
    • Інформація про фоновий процес виводиться справа на панелі сторінки "Додати продукти"
    • Результат виконання фонового процесу заноситься у модель (BackgroundProcessMessage) повідомлень про результат виконання. (Кількість запитів(Успішних, Збійних), Результати вставки у таблицю)
    • Після виконання фонового процесу, результат виводиться під полем вводу потрібних ID із сайту Розетки, рядки що вивелись із Таблиці видвляються
    • Наступне поновлення сторінки витирає ці повідомлення
  4. Сторінка ("Мої продукти"), на якій вивести всі продукти, які є в системі, із зазначенням їхніх основних параметрів (назва, ціна)
  5. Сторінка "Продукт", (до якої переходити із сторінки "Мої продукти") на ній виводяться вся зібрана інформація про обраний продукт:
    • ІД,
    • назва,
    • короткий опис (якщо є),
    • категорія,
    • бренд,
    • ціна,
    • стара ціна (якщо є),
    • лінка на продукт (на сайті Розетки)
    • зображення продукта (лінк на зображення на сайті Розетки)[може завантажитись не зразу]
